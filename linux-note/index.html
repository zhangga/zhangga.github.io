  1. 使用git checkout撤销本地修改，仅限未add/commit的

    git checkout .

    git checkout [filename]

    git checkout . && git clean -xdf

  2. 使用git reset回退项目版本，可以回退掉已add/commit的

    git reset --hard [commit-hashcode]
